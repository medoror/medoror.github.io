Git Cheat Sheet

2019-01-20

Git is absolutely essential knowledge if you want to be a developer. If you don’t know get familiar. For more of my take on git workflow and configurations see my other post. Below is my personal cheat sheet and useful commands I have come across

 

 

Useful Commands

Create a branch and switch to it with local changes

git checkout -b <branch-name> origin/<branch>

Push created feature branch to the remote

git push -u origin <branch-name>

Squash commits

git rebase -i HEAD~<number of commits to squash>

Grabs a commit from another branch

git cherry-pick <SHA>

Add all untracked files

git add -A

Commit all files

git commit -a

Fix commit message or commit

git commit –amend

Useful commit to grabbing a range of commits to place into current branch using a recursive strategy. Strategies can be used for git merges and pulls. More info can be found here

git cherry-pick –strategy=recursive -X theirs <oldest-sha-included>^..<newest-sha>
